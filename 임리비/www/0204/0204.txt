목록을 나타내는 요소

1. 항목1
2. 항목2
3. 항목3
4. 항목4

ol (숫자 항목 요소_순서가 있는 목록)
   li  항목1   /li (필수 자녀요소)
   li  항목2  /li 

+ <ol start-""> : 숫자 시작점 정하기.  ex)<ol start-"31"> : 31부터
  + <ol start-"" reversed> : 숫자 거꾸로  ex) 31,30,29


ul (점 항목 요소_순서가 없는 목록)
   li  항목1   /li
   li  항목2   /li
   li  항목3  /li

dl (정의와 내용 요소)
     dt(정의)
     dd(내용)

  dl> dt+dd
  dl> dt dt +dd       모두 사용 가능
  dl> dt+dd dd

1. ol요소는 순서가 있는 목록
2. ul요소는 순서가 없는 목록
3. ol, ul 요소의 자식으로는 li요소만 올 수 있음
4. li 요소는 블럭 요소로 자신을 포함한 대부분의 블럭, 인라인 요소를 가질 수 있다.
5. dl 요소는 개념, 정의를 나타내는 목록이다.
6. dl 요소는 자식으로 dt, dd 요소만 올 수 있다.
7. dt 요소는 개념을 나타낸다.
8. dd 요소는 정의를 나타낸다.
9. dt, dd는 1:1 뿐 아니라 다:다 의 형식도 가질 수 있다.

스타일시트

- 첫 글자는 반드시 알파벳이어야 한다.
- 두 번째 글자 부터는 알파벳, 숫자, 특수문자 중 - _ 사용 가능
- 대소문자를 구분한다.
- 일반적으로 html문 안에 쓰는 대부분의 알파벳은 소문자를 사용한다.
- div는 의미없는 블럭요소
- span은 의미없는 인라인요소
- class 속성은 모든 요소에 넣을 수 있는 속성, 주로 css, script 를 위해 사용된다.
- id 값은 페이지 내에 단 한번만 사용 가능하나, class 값은 여러번 선언 가능하다.
- class 값은 띄어쓰기를 통해서 여러개의 값 지정이 가능하다.
- class, id 값은 무조건 짧게 하기 보다는 알기 쉽고, 기억에 남도록 의미에 맞게 넣는 것이 좋다.
- id는 페이지 내에 단 한번만 선언 가능하므로 유일한 이름을 짓는 것이 적합하다.
